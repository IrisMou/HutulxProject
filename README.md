# 扬热竞技-专业VR竞技联盟

  应用介绍: "扬热竞技-专业VR竞技联盟"app是一款购买VR商品和预约VR馆的应用。

  核心功能: VR(预约+商城)。
  
    商城：VR进阶书籍、VR眼镜、VR座椅等；
         核心流程：app内下单 - 凭订单编号 - 线下到店取货。
          
    场馆：VR场馆；
          预定后需在预约日期到达VR场馆进行验码消费，逾期订单将自动取消哦。
          
    首页：VR商品在线、VR场馆预定、人气爆款热销单品、优选VR馆和VR单品必买等。
    
    VR订单：商品订单(VR商品下单后，订单列表)。

    VR馆预约：全部预约、待消费、已消费。
    
    我的：商品收藏、VR馆收藏以及相关设置等。
